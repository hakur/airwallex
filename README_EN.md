## Quick Start

```go
package main

import (
    "context"
    "log"
    "time"

    "github.com/hakur/airwallex"
    "github.com/hakur/airwallex/sdk"
)

func main() {
    // Sandbox environment (recommended for dev/test)
    client := airwallex.New("your-client-id", "your-api-key",
        sdk.WithBaseURL(sdk.SandboxURL))

    // Or load from .env file
    // client, _ := airwallex.NewFromEnv(".env", sdk.WithBaseURL(sdk.SandboxURL))

    ctx, cancel := context.WithTimeout(context.Background(), 10*time.Second)
    defer cancel()

    // Query balances
    balances, err := client.Core().GetCurrentBalances(ctx)
    if err != nil {
        log.Fatal(err)
    }
    for _, b := range balances {
        log.Printf("%s: %s", b.Currency, b.AvailableAmount)
    }
}
```

## Multi-Merchant (Platform)

```go
// Client-level: all requests carry x-on-behalf-of
client := airwallex.New(id, key, sdk.WithOnBehalfOf("acct_123"))

// Request-level: override default
client.Payouts().CreateTransfer(ctx, req, sdk.WithRequestOnBehalfOf("acct_456"))
```

## Key Conventions

- **Monetary amounts**: use `string` type (API returns string-formatted amounts like `"100.00"`)
- **Exchange rates**: use `float64` type
- **Timestamps**: use `string` type, never `time.Time`
- **Enums**: use `type Status = string` (alias), not `type Status string`
- **Tests**: use testify `assert`/`require`, no `t.Skip`, no bare `if err != nil`
- **Request IDs**: `"prefix-" + time.Now().Format("20060102150405")`
